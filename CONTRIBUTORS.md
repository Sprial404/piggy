## How to Become a Contributor

We welcome contributions from everyone! Here's how you can become a contributor:

### Types of Contributions

We value all kinds of contributions, including:

- **Code contributions** - Bug fixes, new features, refactoring, performance improvements
- **Documentation** - Improvements to README, guides, docstrings, examples
- **Bug reports** - Detailed bug reports with reproduction steps
- **Feature suggestions** - Well-thought-out proposals for new functionality
- **Code reviews** - Reviewing pull requests and providing feedback
- **Testing** - Writing tests, testing features, reporting issues

### Getting Listed

You will be added to this file when you:

1. Have a pull request merged
2. Contribute significant documentation improvements
3. Provide substantial help with testing or bug reports
4. Participate actively in project discussions and planning

Please see [CONTRIBUTING.md](CONTRIBUTING.md) for details on how to contribute.
